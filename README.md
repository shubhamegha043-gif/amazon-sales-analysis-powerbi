# 📊 Amazon Sales Analysis (Power BI)
Interactive Power BI dashboard analyzing Amazon sales performance and business insights


## 📌 Project Overview

This project presents an interactive Power BI dashboard to analyze Amazon sales data and uncover key business insights. The dashboard helps in understanding sales performance, customer behavior, and product trends.

---

## 🎯 Objectives

* Analyze overall sales performance
* Identify top-selling products
* Track revenue trends over time
* Understand regional sales distribution
* Analyze customer purchasing behavior

---

## 🛠 Tools Used

* Power BI
* Data Visualization
* Data Cleaning & Transformation

---

## 📂 Project Files

* Amazon Sales Dashboard.pbix

---

## 📊 Dashboard Features

* 📈 Sales trend analysis (monthly/yearly)
* 🌍 Region-wise sales performance
* 🛒 Top-selling products & categories
* 💰 Revenue and profit insights
* 🔍 Interactive filters and slicers
